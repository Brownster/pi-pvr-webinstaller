            Frequently Asked Questions
          

          

            

              
How do I add a new media library?

              

                
To add a new media library:

                
    
                  
  1. Go to the Storage tab and ensure your media drive is properly mounted
    2. 
                  
  2. Create a new folder for your media type
    3. 
                  
  3. In your media server (Jellyfin/Plex/Emby), add the new library pointing to this folder
    4. 
                

              

            

            
            

              
How do I update PI-PVR?

              

                
To update PI-PVR:

                
    
                  
  1. Go to the Settings tab and click "Check for Updates"
    2. 
                  
  2. If an update is available, click "Update Now"
    3. 
                  
  3. The system will download and apply the update automatically
    4. 
                

                
Alternatively, you can run ./pi-pvr.sh --update from the command line.

              

            

            
            

              
Why can't I connect to my VPN?

              

                
Common VPN connection issues:

                
    
                  
  • Incorrect credentials - Check your username and password
    • 
                  
  • Server issues - Try selecting a different server region
    • 
                  
  • Network issues - Ensure your network allows VPN connections
    • 
                  
  • Outdated VPN container - Try updating the Gluetun container
    • 
                

                
Check the VPN logs for detailed error messages.

              

            

            
            

              
How do I access my media server remotely?

              

                
There are several ways to access your media server remotely:

                
    
                  
  1. Tailscale (Recommended): Enable Tailscale in the Network tab for secure access without port forwarding
    2. 
                  
  2. Port Forwarding: Configure your router to forward the media server port (e.g., 8096 for Jellyfin)
    3. 
                  
  3. Reverse Proxy: Set up Nginx Proxy Manager with a domain name for secure access
    4. 
                

              

            

            
            

              
How do I add more storage?

              

                
To add more storage to your PI-PVR setup:

                
    
                  
  1. Connect your new storage device to your system
    2. 
                  
  2. Format the drive with a compatible filesystem (ext4 recommended)
    3. 
                  
  3. Go to the Storage tab and click "Mount New Drive"
    4. 
                  
  4. Select the drive and specify a mount point
    5. 
                  
  5. Update your media server paths to include the new storage location
    6.
